# 项目进度管理

类似 Monday.com 的轻量级项目管理网站，使用 React + Vite 构建。支持任务编辑、本地持久化与基础登录验证。

## 功能

- **账号**：注册 / 登录（邮箱 + 密码，至少 6 位）
- **任务字段**：任务名称、任务明细、完成百分比、预定开始时间、截止时间 (DDL)
- **任务操作**：新建、编辑、删除
- **持久化**：用户与任务数据保存在浏览器 localStorage，刷新不丢失
- **统计**：总任务数、已完成数、逾期数、平均进度

## 使用

```bash
npm install
npm run dev
```

浏览器打开提示的本地地址（如 `http://localhost:5173`），先注册再登录，即可在仪表盘管理任务。

## 技术栈

- React 19 + TypeScript
- Vite 7
- React Router
- 无后端，仅 localStorage

## 说明

- 密码为简单 base64 存储，仅适合个人本地使用，不可用于生产环境。
- 数据仅存在当前浏览器，清除站点数据会丢失。
